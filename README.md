# CipherApp

A Python application that encodes and decodes text messages. It shifts ASCII characters using a user-provided key and maps them into 2D coordinates. The application can also generate a visual plot of the encoded message's path[cite: 1].

## Features
* **Encode**: Converts a text message and a numeric key into a list of X and Y coordinates[cite: 1].
* **Decode**: Reads a text file of coordinates and a key to convert them back into the original message[cite: 1].
* **Visualization**: Plots the path of the coordinates using a generated color gradient and saves the graph as a PNG image[cite: 1].

## Installation
1. Install the required dependencies:
   `pip install -r requirements.txt`

## Usage
1. Run the main script:
   `python main.py`
2. Follow the on-screen menu to choose between encoding, decoding, or exiting the application[cite: 1].
